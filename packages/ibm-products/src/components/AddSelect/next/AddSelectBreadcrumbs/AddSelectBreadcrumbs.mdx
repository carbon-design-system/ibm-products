# Add Select Breadcrumbs

## Overview

The Add Select Breadcrumbs component provides hierarchical navigation for the
add-select pattern. It shows the current navigation path and allows users to
navigate back to previous levels.

## Features

- Shows the current navigation path
- Clickable breadcrumb items for navigation
- Automatically truncates long paths
- Accessible keyboard navigation
- Integrates with Carbon's Breadcrumb component

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import { AddSelect } from '@carbon/ibm-products';
```

  setPath(path.slice(0, index + 1));

## Props

### path

Array of breadcrumb items, each containing:

- **id** (string): Unique identifier
- **title** (string): Display text

### onBreadcrumbClick

Callback function when a breadcrumb is clicked:

```typescript
(index: number) => void
```

The index parameter indicates which breadcrumb was clicked (0-based).

## Accessibility

The component follows WCAG 2.1 AA guidelines:

- Uses Carbon's accessible Breadcrumb component
- Supports keyboard navigation
- Provides proper ARIA labels
- Clear visual indication of current page
