## Overview

The Add Select List component manages the list of items and handles keyboard
navigation. It provides:

- Single or multi-select support (checkbox or radio button)
- Keyboard navigation support (arrow keys, home, end)
- Focus management
- Accessible grid structure

### Getting started

Here's a quick example to get you started.


### Keyboard Navigation

The AddSelectList component supports full keyboard navigation:

- **Arrow Down**: Move focus to next item
- **Arrow Up**: Move focus to previous item
- **Arrow Right**: Navigate to children (if item has children)
- **Enter/Space**: Select/deselect the focused item
- **Ctrl+Home**: Move focus to first item
- **Ctrl+End**: Move focus to last item

### Accessibility

The component follows WCAG 2.1 AA guidelines:

- Uses proper ARIA roles (`grid`, `row`, `gridcell`)
- Supports keyboard navigation
- Provides proper focus management
- Uses semantic HTML elements
- Includes proper labels and descriptions
