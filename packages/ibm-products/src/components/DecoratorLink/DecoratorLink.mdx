## Example **setLabelTitle()**

```js
setLabelTitle: (score, scoreThresholds, magnitude) => {
  if (typeof score !== 'number') {
    return 'Unknown score';
  }
  return `"${magnitude}" magnitude. Score ${score} out of ${
    scoreThresholds[scoreThresholds.length - 1]
  }`;
},
```
