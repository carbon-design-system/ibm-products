Title bar modules provide interchangeable and reliable headings for establishing
consistent hierarchies.

### Component API

<ArgsTable />

_**\***Required property_

Additional props passed into `TitleBarModule` will be forwarded along to the
underlying title bar module containers.

#### `label`

Use the `label` prop to specify whether a label should be used.

<Canvas withSource="none">
  <Default
    label="Label"
    element="h2"
    title="Section title"
    parameters={parameters}
  />
</Canvas>

```jsx
<TitleBarModule title="Section title" label="Label" />
```

#### `labelDirection`

Use the `labelDirection` prop to position the label above or below the title.

<Canvas withSource="none">
  <Default
    label="Label"
    labelDirection="bottom"
    element="h2"
    title="Section title"
    parameters={parameters}
  />
</Canvas>

```jsx
<TitleBarModule title="Section title" label="Label" labelDirection="top" />
```

#### `subsection`

Use the `subsection` prop to specify whether a subsection should be used.

<Canvas withSource="none">
  <Default title="Sub-section title" subsection parameters={parameters} />
</Canvas>

```jsx
<TitleBarModule title="Sub-section title" subsection />
```

#### `element`

This prop specifies a different element to be rendered as a title.

<Canvas withSource="none">
  <Default element="h3" title="Section 3 title" parameters={parameters} />
</Canvas>

```jsx
<TitleBarModule element="h3" title="Section 3 title" />
```

### Anatomy

1. **Title:** This the title itself, learn more about potential
   [variants](#variants) above.
2. **Top spacer:** There is 16px spacer `$spacing-05`, included at the top of
   the TitleBarModule.
3. **Bottom spacer:** The 16px spacer `$spacing-05`, included inside the bar,
   means that the module can be placed flush to edge of the components below.

### With Label

A label can be positioned above or below the title using `labelDirection`.

<img alt="`TitleBarModule` withLabel" src={withLabel} />

```jsx
<TitleBarModule title="Section title" label="Label" labelDirection="top" />
```

### With Actions

Actions can be placed inline with the title.

The title bar can accept two type variants, sub-section title and section title,
so you shouldn't supplement your own type style.

<img alt="`TitleBarModule` withActions" src={withActions} />

```jsx
<TitleBarModule title="Sub-section title" subsection />
```

### With Actions and Label

The label can be applied to both the section title and `subsection` title, and
can be positioned above or below the title with actions.

<img alt="`TitleBarModule` withActionsAndLabel" src={withActionsAndLabel} />

```jsx
<TitleBarModule title="Section title" label="Label" subsection />
```

### Grid placement

Title bar modules span a minimum of 4 columns, following the
[narrow grid](https://react.carbondesignsystem.com/?path=/docs/grid--narrow)
implementation.

<Row>
  <Column>
    <img alt="`TitleBarModule` do" src={doImage} />
  </Column>

  <Column>
    <img alt="`TitleBarModule` don’t" src={dont} />
  </Column>
</Row>

```jsx
<Column lg={4}>
  <TitleBarModule title="Section title" />
</Column>
```
