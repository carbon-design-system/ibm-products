# Add Select Row

## Overview

The Add Select Row component represents an individual row in the list. It
provides:

- Single or multi-select support (checkbox or radio button)
- Optional subtitle for additional context
- Optional metadata slot for tags or badges
- Hierarchical navigation support with chevron indicator
- Disabled state
- Keyboard accessible

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import { AddSelect } from '@carbon/ibm-products';
```

## Props

### Required Props

- **itemId** (string): Unique identifier for the item
- **title** (string): Item title
- **value** (string): Item value

### Optional Props

- **subtitle** (string): Item subtitle for additional context
- **selected** (boolean): Whether the item is selected
- **disabled** (boolean): Whether the item is disabled
- **hasChildren** (boolean): Whether the item has children (shows navigation
  indicator)
- **parentId** (string): Parent ID for hierarchical navigation
- **icon** (ReactNode): Optional icon slot
- **meta** (ReactNode): Optional metadata slot (e.g., tags, badges)
- **className** (string): Optional class name
