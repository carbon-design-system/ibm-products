## Overview

The Add Select Breadcrumbs component provides hierarchical navigation within the
Add Select interface. It displays the current navigation path and allows users
to navigate back to previous levels by clicking on breadcrumb items.

### Features

- Displays current navigation path
- Clickable breadcrumb items for navigation
- Automatically highlights current page
- Supports deep navigation hierarchies
- Integrates with Carbon Breadcrumb component

### Getting started

Here's a quick example to get you started.

#### JS (via import)

```javascript
import '@carbon/ibm-products-web-components/es/components/add-select/index.js';
```

#### Usage Example

<Canvas className="page-header--docs-demo" of={AddSelectBreadcrumbsStories.Default} />

```javascript
const path = [
  { id: 'root', title: 'All Items' },
  { id: 'category-1', title: 'Category 1' },
  { id: 'subcategory-1', title: 'Subcategory 1' },
];

const handleBreadcrumbClick = (event) => {
  const { index } = event.detail;
  // Navigate back to the clicked level
  console.log('Navigated to:', path[index].title);
};
```

```html
<c4p-add-select>
  <c4p-add-select-breadcrumbs
    .path=${path}
    @c4p-add-select-breadcrumbs-click=${handleBreadcrumbClick}
  >
  </c4p-add-select-breadcrumbs>
</c4p-add-select>
```

#### Deep Navigation

The breadcrumbs component supports deep navigation hierarchies with multiple
levels.

<Canvas className="page-header--docs-demo" of={AddSelectBreadcrumbsStories.DeepNavigation} />

```javascript
const path = [
  { id: 'root', title: 'All Items' },
  { id: 'category-1', title: 'Electronics' },
  { id: 'subcategory-1', title: 'Computers' },
  { id: 'subcategory-2', title: 'Laptops' },
  { id: 'subcategory-3', title: 'Gaming Laptops' },
];
```

```html
<c4p-add-select>
  <c4p-add-select-breadcrumbs
    .path=${path}
    @c4p-add-select-breadcrumbs-click=${handleBreadcrumbClick}
  >
  </c4p-add-select-breadcrumbs>
</c4p-add-select>
```
