# Import Modal

The Import Modal pattern is made with carbon design system's web components,

Note: This is not a component, and is only a recipe to build a Import Modal using carbon components meant to be scaled as per specific requirements.

Ingredients used

These are the ingredients used to generate the import modal
cds-modal, cds-modal-header, cds-modal-body, cds-file-uploader, cds-file-uploader-drop-container, cds-form-item, cds-text-input, cds-button, cds-file-uploader-item, and cds-modal-footer.
