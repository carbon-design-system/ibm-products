## Overview

The AddSelectSelectionSummaryPanel component displays a list of selected items
with a count badge and optional edit icon. It supports custom empty states,
custom item rendering, and can accept SelectionSummaryPanelItem components as
children for full control.

### Getting started

Here's a quick example to get you started.

#### JS (via import)

```javascript
import { preview__AddSelect as AddSelect } from '@carbon/ibm-products';
```

#### Usage Example

<Canvas of={stories.Default}></Canvas>

```jsx
<AddSelect.SelectionSummaryPanel
  title="Selected items"
  selectedItems={items}
  showCount
/>
```

## With Edit Icon

Add an edit icon next to the count badge for quick access to edit functionality.

<Canvas of={stories.WithEditIcon}></Canvas>

```jsx
<AddSelect.SelectionSummaryPanel
  title="Selected items"
  selectedItems={items}
  showCount
  showEditIcon
  onEdit={handleEdit}
  editIconDescription="Edit selections"
/>
```

## With Empty State

Provide a custom empty state component when no items are selected.

<Canvas of={stories.WithEmptyState}></Canvas>

```jsx
<AddSelect.SelectionSummaryPanel
  title="Selected items"
  selectedItems={[]}
  emptyState={
    <NoDataEmptyState
      title="No items selected"
      subtitle="Select items to see them here"
    />
  }
/>
```

## With Custom Rendering

Use the `renderItem` prop to customize how each item is displayed.

<Canvas of={stories.WithCustomRendering}></Canvas>

```jsx
<AddSelect.SelectionSummaryPanel
  title="Selected items"
  selectedItems={items}
  renderItem={(item) => (
    <div>
      <strong>{item.title}</strong>
      <div>{item.value}</div>
    </div>
  )}
/>
```
