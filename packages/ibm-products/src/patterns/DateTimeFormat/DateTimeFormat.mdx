# Date Time Format

`dateTimeFormat` is a utility from `@carbon/utilities` that wraps the browser
internationalization APIs for formatting relative and absolute date and time
values.

## Overview

Use this utility when you need consistent formatting for:

- absolute date and time values
- date-only values
- time-only values
- relative timestamps such as “3 minutes ago”

site and shows how locale and format selections affect the rendered output.
