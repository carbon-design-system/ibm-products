# Add Select Body

## Overview

The Add Select Body component contains the main content area for the add-select pattern. It includes:
- Global search functionality
- Breadcrumb navigation for hierarchical data
- Item count display
- Content area for the item list

This component manages the search state and navigation path, and emits events when users interact with search or breadcrumbs.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibm-products-web-components/es/components/add-select/index.js';
```

### Usage Example

## Events

### `c4p-add-select-body-search`
Fired when the search term changes.

**Detail:**
```javascript
{
  searchTerm: string
}
```

### `c4p-add-select-body-breadcrumb-click`
Fired when a breadcrumb item is clicked.

**Detail:**
```javascript
{
  index: number
}
```
