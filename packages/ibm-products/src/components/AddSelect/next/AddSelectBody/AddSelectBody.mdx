# Add Select Body

## Overview

The Add Select Body component contains the main content area for the add-select
pattern. It includes:

- Global search functionality
- Breadcrumb navigation for hierarchical data
- Item count display
- Content area for the item list

This component manages the search state and navigation path, and provides
callbacks when users interact with search or breadcrumbs.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import { AddSelect } from '@carbon/ibm-products';
```

## Events

### `onSearch`

Callback when the search term changes.

**Parameters:**

```typescript
(searchTerm: string) => void
```

### `onBreadcrumbClick`

Callback when a breadcrumb item is clicked.

**Parameters:**

```typescript
(index: number) => void
```
