## Overview

The Add Select Body component contains the main content area for the add-select
pattern. It provides:

- Global search functionality
- Breadcrumb navigation for hierarchical data
- Item count display with tag
- Content area for the item list
- Search state management
- Callbacks for search and breadcrumb interactions

### Getting started

Here's a quick example to get you started.

#### JS (via import)

```javascript
import '@carbon/ibm-products-web-components/es/components/add-select/index.js';
```

#### With Breadcrumbs

Display breadcrumb navigation for hierarchical data structures.

<Canvas className="page-header--docs-demo" of={AddSelectBodyStories.WithBreadcrumbs} />

```javascript
const path = [
  { id: 'root', title: 'Root' },
  { id: 'category', title: 'Category' },
  { id: 'subcategory', title: 'Subcategory' },
];

const handleBreadcrumbClick = (event) => {
  const { index } = event.detail;
  // Navigate back to the clicked level
  console.log('Navigate to:', path[index].title);
};
```

```html
<c4p-add-select>
  <c4p-add-select-body
    items-label="All items"
    global-search-label="Search"
    global-search-placeholder="Search items"
    .path=${path}
    @c4p-add-select-body-breadcrumb-click=${handleBreadcrumbClick}
  >
    <c4p-add-select-list>
      <!-- Add select items here -->
    </c4p-add-select-list>
  </c4p-add-select-body>
</c4p-add-select>
```

### Events

#### `c4p-add-select-body-search`

Callback when the search term changes.

```typescript
(event: CustomEvent<{ searchTerm: string }>) => void
```

#### `c4p-add-select-body-breadcrumb-click`

Callback when a breadcrumb item is clicked.

```typescript
(event: CustomEvent<{ index: number }>) => void
```
