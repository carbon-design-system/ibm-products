## Overview

The AddSelectSidePanel component is an optional wrapper that provides consistent
layout and styling for the side panel area. It supports different width variants
and serves as a container for other side panel components like
SelectionSummaryPanel and ItemInfoPanel.

### Getting started

Here's a quick example to get you started.

#### JS (via import)

```javascript
import { preview__AddSelect as AddSelect } from '@carbon/ibm-products';
```

#### Usage Example

<Canvas of={stories.Default}></Canvas>

```jsx
<AddSelect.SidePanel width="default">
  <AddSelect.SelectionSummaryPanel {...props} />
</AddSelect.SidePanel>
```

## Width Variants

The side panel supports three width variants:

- **narrow** (16rem) - For compact side panels
- **default** (22.5rem) - Standard width, matches legacy implementation
- **wide** (28rem) - For side panels with more content
