# Add Select

## Overview

The Add Select component is a wrapper component that provides the container for the add-select pattern. It should contain the `c4p-add-select-body` component which handles the main functionality including search, breadcrumbs, and item lists.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibm-products-web-components/es/components/add-select/index.js';
```

### Usage Example

<Canvas className="page-header--docs-demo" of={AddSelectStories.Default} />

### HTML

```html
<c4p-add-select>
  <c4p-add-select-body
    items-label="All items"
    global-search-label="Search"
    global-search-placeholder="Search items"
  >
    <c4p-add-select-list>
      <c4p-add-select-item
        item-id="1"
        title="Item 1"
        subtitle="Description"
        value="item-1"
      >
      </c4p-add-select-item>
    </c4p-add-select-list>
  </c4p-add-select-body>
</c4p-add-select>
```
