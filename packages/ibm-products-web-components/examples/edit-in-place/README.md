# EditInPlace Example

This example demonstrates the usage of the EditInPlace component from Carbon for IBM Products Web Components.

## Getting Started

### Prerequisites

- Node.js (version specified in `.nvmrc` at the repository root)
- Yarn

### Installation

From the repository root, run:

```bash
yarn install
```

### Running the Example

From this directory, run:

```bash
yarn dev
```

This will start a local development server. Open your browser to the URL shown in the terminal (typically `http://localhost:5173`).

## What's Included

This example showcases:

- **Default EditInPlace**: Basic inline editing functionality
- **Invalid State**: Shows validation error state
- **Read Only**: Demonstrates read-only mode with toggletip
- **Large Size**: Example with larger size variant

## Features Demonstrated

- Inline text editing
- Save and cancel actions
- Validation states
- Read-only mode
- Different size variants
- Event handling (check browser console for event logs)
