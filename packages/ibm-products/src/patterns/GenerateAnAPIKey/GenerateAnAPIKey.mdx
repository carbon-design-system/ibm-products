## Overview

When designing for the generation of API keys, ensure users know what the key is
created for, the security implications, and the end destination of the key.

To build this pattern, we recommend including the following ingredients:

- [ComposedModal](https://react.carbondesignsystem.com/?path=/docs/components-composedmodal)
- [cds-button](https://react.carbondesignsystem.com/?path=/docs/components-button)
- [TextInput](https://react.carbondesignsystem.com/?path=/docs/components-textinput)
- [PasswordInput](https://react.carbondesignsystem.com/?path=/docs/components-passwordinput--overview)

```jsx
import { Button, Modal, TextInput, PasswordInput } from '@carbon/react';
```
