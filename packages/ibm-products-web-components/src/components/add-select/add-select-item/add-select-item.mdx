# Add Select Item

## Overview

The Add Select Item component represents a single selectable item in the add-select pattern. It supports:
- Single-select mode (radio button)
- Multi-select mode (checkbox)
- Hierarchical navigation (items with children)
- Disabled state
- Selected state
- Keyboard navigation

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibm-products-web-components/es/components/add-select/index.js';
```

### Usage Example

<Canvas className="page-header--docs-demo" of={AddSelectItemStories.Default} />

## Events

### `c4p-add-select-item-select`
Fired when the item is selected or deselected.

**Detail:**
```javascript
{
  itemId: string,
  selected: boolean,
  value: string
}
```

**Example:**
```javascript
const handleSelect = (event) => {
  const { itemId, selected, value } = event.detail;
  console.log(`Item ${itemId} is now ${selected ? 'selected' : 'deselected'}`);
};
```

### `c4p-add-select-item-navigate`
Fired when navigating to children (only for items with `has-children` attribute).

**Detail:**
```javascript
{
  itemId: string,
  title: string,
  parentId: string
}
```

**Example:**
```javascript
const handleNavigate = (event) => {
  const { itemId, title } = event.detail;
  console.log(`Navigating to children of ${title}`);
  // Load and display children items
};
```

## Keyboard Navigation

- **Enter/Space**: Select/deselect the item
- **Arrow Right**: Navigate to children (if `has-children` is true)
