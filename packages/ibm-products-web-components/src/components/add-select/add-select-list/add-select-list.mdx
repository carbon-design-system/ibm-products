# Add Select List

## Overview

The Add Select List component is a container for add-select items. It provides the structure for displaying a list of selectable items in either single-select or multi-select mode. The list uses a treegrid role for accessibility and keyboard navigation support.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon/ibm-products-web-components/es/components/add-select/index.js';
```

## Accessibility

The list uses the `treegrid` role for proper accessibility support. Each item should have a `tab-index` attribute for keyboard navigation. The first item in the list should have `tab-index="0"`, while subsequent items should have `tab-index="-1"` to support roving tabindex pattern.
